Python Django - Build Complex MultiVendor Ecommerce System
---------

>pip freeze
>virtualenv --version
>python -m venv venv ( Creating Virtual Environment)
>venv\scripts\activate(To activate the Virtual Environment)
>venv\scripts\deactivate(To deactivate the virtual Environment)
>pip install django
>django-admin --version (To check th Django Version (5.1.3))
>python.exe -m pip install --upgrade pip ( To upgrade pip)
>django-admin startproject carzone .(To create the django Project)
>python manage.py runserver ( To run local server)
>python manage.py startapp pages(To create pages app, Created apps need to configure in setting.py INSTALLED_APPS Section (like ('pages.apps.PagesConfig'))

   STATIC_ROOT = os.path.join(BASE_DIR, 'static')
   STATICFILES_DIRS=[
    os.path.join(BASE_DIR, 'carzone/static'),
]
